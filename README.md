Overview

Assignment 5 focuses on building a Task Manager App using React Native and Expo.
The app demonstrates the use of Expo Router, Context API, component-based UI, and multi-screen navigation.
Users can add tasks, view task details, and mark tasks as completed using a clean tab-based interface.

Features

Add new tasks with title and description

View detailed information for each task

Mark tasks as completed

Separate tabs for All Tasks and Completed Tasks

Global state management using Context API

Reusable TaskItem component

Simple and organized UI following React Native best practices
